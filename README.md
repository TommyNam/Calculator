## Project Overview:
Calculator Project


## Features
- Operator Buttons (+, -, /, *)
- Numeric Buttons (0-9)
- Equal button calculates user-input
- Odd/Even Button (Allows numbers to be odd)
- Music Button (Plays Naruto Main Theme)
- Clear Entry (Clears display)
- Clear (Clears last user input || Backspace) 


## JS Validations
- No double operators (++ or */ cannot be used)
- Only one decimal per number
- Behavior after getting a result (Allows user to use result and continue equation)
- Sign Toggle Safety (+/- does nothing if the display is empty)
- Error message if something cant be calculated
